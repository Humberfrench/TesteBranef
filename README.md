# Teste Branef

Teste usando DDD, EF, Unit of Work, ValidationResult, Repository, Dependency Injection.

## Cadastro de Clientes

O sistema de cadastro de clientes contém os seguintes campos:

- **ID**
- **Nome da Empresa**
- **Telefone da Empresa**
- **Email da Empresa**
- **Porte da Empresa**
  - O porte da empresa contém 3 valores:
    - Pequena (P)
    - Média (M)
    - Grande (G)
